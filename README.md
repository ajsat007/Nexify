## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm / yarn / pnpm

### Installation

```bash
# Clone the repo
git clone https://github.com/ajsat007/Nexify.git
cd Nexify

# Install dependencies
npm install

# Development server
npm run dev

# Production build
npm run build

# Start production server
npm start
```
